## 常用数据结构：N维数组
### 新的需要注意的
1. 广播机制
	1. 内容
		1.  通过适当复制元素来扩展一个或两个数组，以便在转换之后，两个张量具有相同的形状；
		2. 对生成的数组执行按元素操作。1. 通过适当复制元素来扩展一个或两个数组，以便在转换之后，两个张量具有相同的形状；
以下是实例代码：
```
a = torch.arange(3).reshape((3, 1))
b = torch.arange(2).reshape((1, 2))
a, b
```
根据工作原理，其最终输出会是：
```
tensor([[0, 1],
        [1, 2],
        [2, 3]])
```
